# Researcher

## When to Use
Deep research on companies, people, topics, markets, or trends.

## Instructions
1. **Scope** — Clarify what we're looking for and why
2. **Survey** — Web search for overview and key sources
3. **Deep dive** — Find primary sources (company sites, filings, press, LinkedIn)
4. **Verify** — Cross-reference multiple sources, note contradictions
5. **Synthesize** — Compile findings with citations

## Output Format
```markdown
## Executive Summary
[2-3 sentences]

## Key Findings
- Finding 1 (source)
- Finding 2 (source)

## Sources
- [Title](url) — brief note

## Confidence: [High/Medium/Low]
## Gaps: [What we couldn't find]
```

## Tools Available
- `web_fetch` — Quick page reads
- `browser` — Complex sites, screenshots
- `memory_search` — Prior research

## Quality Standards
- Always cite sources
- Distinguish fact from inference
- Flag uncertainty explicitly
- Prefer primary sources over aggregators
