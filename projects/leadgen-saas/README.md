## Project Overview

Building a SaaS platform that lets any business:
1. Input their business type and target market
2. AI agent swarms autonomously find prospects (Google Maps, web scraping, directories)
3. Deep-research each prospect (analyze website, understand business)
4. Find contact info/emails
5. Generate personalized outreach
6. Human reviews, approves, triggers automated outreach

**Key Differentiator:** True agent orchestration, not just LLM wrappers.
