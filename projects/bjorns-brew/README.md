## Overview

Dog-friendly coffee shop website built with Next.js 15, featuring their signature Pup Cup and community-focused branding.

## What We Built

- **Next.js 15** site with App Router
- **Tailwind CSS** styling
- Dog-centric design with paw print accents
- Floating Bjorn hero with "$183K raised" badge
- Menu page with milk alternatives banner
- Featured Pup Cup section (FREE for furry friends)
- Impact gallery + charity partners
- "Pups of Bjorn's" community section
- Seasonal banner (Winter Spiced Mocha)
- Mobile-responsive design
