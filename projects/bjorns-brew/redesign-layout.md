### 1.3 Row Structure

**8-Row Grid** for vertical organization

| Row | Height % | Purpose |
|-----|----------|---------|
| Row 1 | 10% | Header/Logo zone |
| Row 2 | 15% | Featured/Hero zone |
| Row 3 | 17.5% | Primary category A |
| Row 4 | 17.5% | Primary category B |
| Row 5 | 15% | Secondary category |
| Row 6 | 12.5% | Modifiers/Add-ons |
| Row 7 | 7.5% | Mission/Footer |
| Row 8 | 5% | Bottom margin/bleed |

**Row Gutter Height:** 1% of total height

### 1.4 Modular Units

For consistent spacing, define a **base unit (BU)**:
- **BU = 0.5 inches** (12.7mm)
- All spacing should be multiples of BU

| Spacing Type | BU Multiple | Actual Size |
|--------------|-------------|-------------|
| Micro | 1 BU | 0.5" |
| Small | 2 BU | 1.0" |
| Medium | 3 BU | 1.5" |
| Large | 4 BU | 2.0" |
| XL | 6 BU | 3.0" |
| Section | 8 BU | 4.0" |

### 2.4 Typography Hierarchy

**Level 1: Feature Title**
- Size: 2.5-3" / 180-216pt
- Weight: Bold/Black
- Tracking: -2%
- Use: Featured item name only

**Level 2: Category Headers**
- Size: 1.5-2" / 108-144pt
- Weight: Bold
- Tracking: 5% (slightly expanded)
- Use: Section titles (BLENDED DRINKS, FOOD)

**Level 3: Item Names**
- Size: 0.75-1" / 54-72pt
- Weight: Medium/Semibold
- Tracking: 0%
- Use: Individual menu items

**Level 4: Descriptions**
- Size: 0.5-0.625" / 36-45pt
- Weight: Regular
- Tracking: 0%
- Use: Item descriptions, modifiers

**Level 5: Prices**
- Size: 0.5-0.625" / 36-45pt
- Weight: Regular (match descriptions)
- Tracking: 0%
- Use: All pricing (NOT in columns)

**Level 6: Fine Print**
- Size: 0.375-0.5" / 27-36pt
- Weight: Regular
- Tracking: 1%
- Use: Footer, disclaimers, mission

## 5. Spacing Rules

### 5.1 Margin Specifications

**Outer Margins (Board Edge to Content)**
- Top: 3% (0.96" on 32" height)
- Bottom: 4% (1.28" on 32" height)
- Left: 2% (0.96" on 48" width)
- Right: 2% (0.96" on 48" width)

**Section Margins (Between Major Sections)**
- Vertical separation: 6 BU (3")
- Use visual divider OR whitespace (not both)

### 5.2 Padding Specifications

**Category Header Padding**
- Above header: 4 BU (2")
- Below header: 2 BU (1")
- Creates clear section start

**Item Padding**
- Between items (vertical): 2 BU (1")
- Between columns (horizontal): 3 BU (1.5")
- Price from item name: 4-6 spaces (not aligned)

**Featured Item Padding**
- All sides: 4 BU (2") minimum
- Creates "island" effect
- Background extends 2 BU beyond text

**Photo Padding**
- Between photos: 2 BU (1")
- Below photos to text: 1 BU (0.5")
- Creates unified gallery feel

### 5.4 Price Placement Rules

**CRITICAL: Never column-align prices**

âŒ WRONG:
```

Vanilla Chai .............. 6.75
Mocha Frappe .............. 6.50

```

```
Vanilla Chai Frappe
Warm spices, creamy  6.75 / 7.75 / 8.75
```

**Price Formatting:**
- No dollar signs (6.00 not $6.00)
- Same font weight as descriptions
- 4-6 character spaces after description
- Consider dropping decimals for whole dollars (6 not 6.00)
- Use middot (Â·) or slash (/) between sizes

### 5.5 Visual Rhythm

**Consistent Vertical Rhythm:**
Every element should align to an invisible baseline grid.

**Baseline unit: 0.25" (6.35mm)**

- Body text: 3 baselines (0.75" line height)
- Section headers: 6 baselines (1.5" line height)  
- Feature text: 8 baselines (2" line height)
- Section gaps: 8-12 baselines

**Horizontal Alignment Points:**
- Left margin: 2% in
- Left column content: 4% in
- Center gutter: 50%
- Right column content: 52% in
- Right margin: 98% in

## 7. Implementation Notes

### 7.1 File Preparation

**For Print Production:**
- Final file: PDF/X-1a or PDF/X-4
- Resolution: 300 DPI minimum
- Color: CMYK
- Bleed: 0.125" on all sides
- Fonts: Outlined or embedded

**For Digital Display:**
- Format: PNG or MP4 (for animation)
- Resolution: Match display native resolution
- Color: sRGB
- Consider motion graphics for featured items

### 7.2 Recommended Fonts

**Primary (Nordic-inspired sans-serif):**
- **Avenir** â€” Clean, geometric, Scandinavian feel
