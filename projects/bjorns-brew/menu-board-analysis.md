### âŒ Issues to Address

#### Layout & Hierarchy
- [ ] **No clear visual hierarchy** â€” Everything has similar visual weight
- [ ] **Golden Triangle ignored** â€” High-margin items not in eye-tracking hotspots
- [ ] **Information overload** â€” Too much competing for attention
- [ ] **No signature item featured** â€” What's THE drink to get?

#### Typography
- [ ] **Multiple font styles** â€” Creates visual chaos
- [ ] **Inconsistent sizing** â€” Headers and items blur together
- [ ] **Small text** â€” Many items hard to read at distance
- [ ] **Mixed alignment** â€” Some centered, some left-aligned

#### Pricing Psychology
- [ ] **Dollar signs everywhere** â€” Triggers "pain of paying"
- [ ] **No price anchoring** â€” Highest prices not strategically placed
- [ ] **Prices compete with items** â€” Same visual weight

#### Color & Contrast
- [ ] **Low contrast in places** â€” Tan on tan hard to read
- [ ] **Red overused** â€” Loses impact
- [ ] **No color coding** â€” Categories don't have visual distinction

#### Menu Engineering
- [ ] **No featured/seasonal section** â€” Missing LTO opportunity
- [ ] **Flavors overwhelming** â€” 22 options create decision paralysis

## Research-Based Recommendations

### Priority 1: Visual Hierarchy
1. Create clear section breaks with color bands
2. Feature 1-2 signature items prominently
3. Use golden triangle for highest-margin items
4. Reduce overall item count displayed

### Priority 2: Pricing Optimization
1. Remove dollar signs (use 6.00 not $6.00)
2. Place highest-priced item first in category (anchoring)
3. De-emphasize prices visually (smaller, lighter weight)
4. Consider removing .00 (just "6" vs "6.00")

### Priority 3: Typography Cleanup
1. Maximum 2-3 font families
2. Clear size hierarchy (Category > Item > Description > Price)
3. Ensure readability at 6-10 foot distance
4. Consistent alignment throughout

### Priority 4: Menu Engineering
1. Add "Featured" or "Bjorn's Pick" callout
2. Group flavors into categories (reduce cognitive load)
3. Add seasonal/LTO section
4. Integrate mission subtly ("$183K donated")

### Priority 5: Food Section
1. Simplify layout â€” fewer items displayed
2. Group by meal occasion (Breakfast, Snacks, etc.)
3. Feature 2-3 hero items with photos
4. Move detailed items to separate menu
