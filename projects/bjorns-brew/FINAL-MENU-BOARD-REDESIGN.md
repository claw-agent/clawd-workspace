# 5. PRICING STRATEGY

## Rules (From Research)

1. **NO dollar signs** ‚Äî Removes "pain of paying" trigger
2. **NO price columns** ‚Äî Don't align prices vertically
3. **Nest prices in copy** ‚Äî Same font as descriptions
4. **Price anchoring** ‚Äî Highest price first in category

## Price Display Format

‚ùå Wrong:
```
Latte Frappe.............$6.00
Chai Frappe..............$6.75
```

‚úÖ Correct:
```
Frozen Velvet Latte  16oz 6.00 ¬∑ 24oz 7.00 ¬∑ 32oz 8.00
Spiced Chai Freeze   16oz 6.75 ¬∑ 24oz 7.75 ¬∑ 32oz 8.75
```
