# Design Team Structure

## Team Lead: Design Director
**Role:** Coordinates all design work, ensures brand consistency, manages design reviews
**Reports to:** Project Manager
**Manages:** UX Lead, UI Lead, Brand Designer

---

## UX Team

### Senior UX Designer (Lead)
**Focus:** User research synthesis, journey mapping, information architecture
**Responsibilities:**
- Define user flows and interaction patterns
- Create wireframes and prototypes
- Conduct usability reviews
- Translate research insights into design requirements

### UX Researcher
**Focus:** User interviews, usability testing, behavioral analysis
**Responsibilities:**
- Design and conduct user research studies
- Analyze qualitative and quantitative data
- Create personas and user journey maps
- Validate design decisions with users

### Interaction Designer
**Focus:** Micro-interactions, animations, state transitions
**Responsibilities:**
- Design interactive prototypes
- Define motion and transition specs
- Ensure accessibility compliance
- Document interaction patterns

---

## UI Team

### Senior UI Designer (Lead)
**Focus:** Visual design systems, component libraries, high-fidelity mockups
**Responsibilities:**
- Create and maintain design system
- Design pixel-perfect interfaces
- Ensure visual consistency across products
- Collaborate with developers on implementation

### Visual Designer
**Focus:** Graphics, illustrations, icons, imagery
**Responsibilities:**
- Create custom illustrations and icons
- Design marketing collateral
- Produce social media visuals
- Support brand consistency

### Motion Designer
**Focus:** Animations, video content, dynamic visuals
**Responsibilities:**
- Create animated content for marketing
- Design loading states and transitions
- Produce demo videos and walkthroughs
- Support interactive prototypes

---

## Brand & Content

### Brand Designer
**Focus:** Brand identity, guidelines, asset management
**Responsibilities:**
- Maintain brand guidelines
- Ensure brand consistency across touchpoints
- Create brand assets and templates
- Support marketing materials

### Content Designer
**Focus:** UX writing, microcopy, content strategy
**Responsibilities:**
- Write clear, effective UI copy
- Define voice and tone guidelines
- Create content templates
- Ensure accessibility in language

---

## Workflow

```
Research Insights → UX Designer → Wireframes
                         ↓
                  Interaction Designer → Prototypes
                         ↓
                    UI Designer → High-fidelity Mockups
                         ↓
                  Visual/Motion → Polish & Assets
                         ↓
                  Content Designer → Copy Integration
                         ↓
                  Design Director → Final Review
```

## Tools
- Figma (primary design tool)
- FigJam (collaboration/whiteboarding)
- Maze (usability testing)
- Loom (design walkthroughs)
