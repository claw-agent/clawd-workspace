## 2. What Separates a $500 Template from a $15K Custom Site

| $500 Template Site | $15K Custom Site |
|---|---|
| Generic stock photos (ladder on roof) | Real crew photos, real completed projects, drone shots |
| Company-centric copy ("We've been in business since...") | Customer-centric copy ("Protect your family from...") |
| One CTA at bottom of page | CTAs every scroll-stop (header, hero, mid-page, footer) |
| Logo badges thrown randomly | Curated trust bar with greyscale badges in a clean horizontal strip |
| Generic "Contact Us" form | "Get Your Free Inspection" with urgency + 3-5 fields max |
| No process visualization | Visual numbered step process (3-5 steps with icons) |
| Reviews on a separate page | Reviews embedded in hero area AND mid-page AND near CTAs |
| One font weight, basic spacing | Typographic hierarchy with 3+ weights, generous whitespace |
| No animations | Subtle scroll-triggered fade-ins, counter animations, hover states |
| Mobile-ignored | Mobile-first with thumb-friendly CTAs, sticky phone button |
| No personality | Founder story, team photos, branded "promise" section |

**Key principles:**
- **Background:** Dark overlay (85-92% opacity) over a high-quality photo of a COMPLETED beautiful home (NOT a construction zone). Alternatively: a subtle video loop of crew working at golden hour
- **Headline:** Customer-pain-focused, NOT company-brag. "Protect Your Family" > "We're The Best Roofers"
- **Bullet points:** 3 max, 10 words or fewer each ‚Äî scannable value props
- **Phone number visible** without scrolling on ALL devices
- **Trust badge bar** directly below hero or at hero bottom ‚Äî greyscale logos for clean look
- **Rating pill** (star + number + review count) above or near headline

**What does NOT work:**
- ‚ùå Sliders/carousels (nobody waits for slide 2)
- ‚ùå Stock photos of people in hard hats pointing at roofs
- ‚ùå Video that autoplays with sound
- ‚ùå "Welcome to [Company Name]" as a headline

### 3.2 Trust Signals

**Quantity:** 8-12 trust elements visible across the homepage  
**Types that work (ranked by conversion impact):**

1. **Google review count + rating** (e.g., "4.9 ‚≠ê from 247 reviews") ‚Äî above the fold
2. **Certification badges** (GAF Master Elite, Owens Corning Preferred, CertainTeed) ‚Äî greyscale strip
3. **BBB A+ rating** with logo
4. **Real customer testimonials** with: reviewer photo, first name + last initial, Google/Yelp logo, star rating, 2-3 sentence quote
5. **Years in business / projects completed** ‚Äî animated counter
6. **"As Seen On" or award logos** ‚Äî if applicable
7. **License & insurance statement** ‚Äî subtle but present
8. **Warranty details** (e.g., "50-Year Warranty on Labor & Materials")
9. **Financing badges** (if offered)
10. **Manufacturer partner logos** (GAF, Owens Corning, etc.)

**Placement pattern:**
- Trust bar (badges): immediately below hero
- Review snippets: 30% down the page
- Full testimonial section: 60% down the page
- Certification details: near footer or about section

### 3.3 CTAs ‚Äî Placement, Wording, Design

**Placement rule: Every 1.5 viewport heights, there should be a CTA.**

| Location | CTA Type | Example Wording |
|----------|----------|----------------|
| Header (sticky) | Phone + Button | `üìû (801) 555-1234` + `Free Estimate` |
| Hero | Primary button + Phone | `Get Your Free Inspection ‚Üí` |
| After services | Inline form | `Schedule Your Free Roof Assessment` |
| Mid-page | Banner | `Storm Damage? We'll Handle Your Insurance Claim` |
| After testimonials | Button | `Join 500+ Happy Homeowners` |
| Footer | Full form + phone + map | `Request Your Free Quote Today` |
| **Mobile sticky** | Bottom bar | `üìû Call Now` + `üí¨ Text Us` |

**Wording patterns that convert (tested across roofing):**
- ‚úÖ "Get Your **Free** Inspection" (the word FREE is critical)
- ‚úÖ "Schedule Your **No-Obligation** Estimate"
- ‚úÖ "See Why 500+ Homeowners Trust Us"
- ‚úÖ "Claim Your Free Storm Damage Report"
- ‚úÖ Urgency: "Limited Availability This Month" or "Before the Next Storm"
- ‚ùå "Submit" (worst performing CTA text in all of marketing)
- ‚ùå "Contact Us" (too vague, no value proposition)

**Button design:**
- Contrasting color (orange/amber on dark blue works great)
- Minimum 48px height on mobile (thumb-friendly)
- Subtle hover: scale(1.03) + shadow increase
- Arrow icon ‚Üí after text adds ~12% more clicks

### 3.4 Color Schemes That Work for Roofing

**The dominant palette pattern across top-converting roofing sites:**

```
Primary:     Deep blue (#1e3a5f to #1e40af) ‚Äî trust, professionalism, sky
Secondary:   Dark charcoal (#1a1a2e to #374151) ‚Äî sophistication, strength  
Accent/CTA:  Orange/Amber (#f97316 to #ea580c) ‚Äî urgency, warmth, action
Background:  White (#ffffff) + Light gray (#f8fafc) alternating sections
Text:        Near-black (#111827) for body, white on dark sections
```

**Why this works:** Blue = trust/sky (roofing). Orange = urgency/action (CTA contrast). The blue-orange complementary pair is proven in home services.

**Alternative palettes for differentiation:**
- **Storm/weather theme:** Slate (#475569) + Electric blue (#3b82f6) + Safety yellow (#eab308)
- **Premium/luxury:** Black (#0f172a) + Gold (#d4a853) + White ‚Äî for high-end roofers
- **Green/eco:** Forest (#166534) + Sage (#86efac) + Warm white ‚Äî for sustainable roofing

**For XPERIENCE Roofing specifically:** Lean into the "X" branding. Consider a bold, modern scheme ‚Äî perhaps dark navy + electric orange/red accent. The "X" could become a recurring design motif (crossed lines, X-shaped section dividers).

### 3.5 Mobile-First Patterns

**72%+ of roofing leads come from mobile.** Design mobile-first, desktop second.

**Must-haves:**
1. **Sticky bottom bar** with Call + Text buttons (always visible)
2. **Click-to-call phone number** in header ‚Äî large enough to tap
3. **Hamburger menu** with CTA button visible even when menu is closed
4. **Single-column layout** ‚Äî no side-by-side cards on mobile
5. **Form fields:** Maximum 4-5 fields. Name, phone, email, service type dropdown, message (optional)
6. **Thumb zone design:** Primary CTAs in the bottom 1/3 of screen
7. **Image optimization:** WebP format, lazy loading, max 200KB per hero image on mobile
8. **Font minimum:** 16px body (prevents iOS zoom), 14px minimum for anything

**Specific mobile patterns:**

### 3.6 Photo & Video Usage

**Photos that convert (specific guidance):**

| Use | Photo Type | Why It Works |
|-----|-----------|-------------|
| Hero background | Completed beautiful home, aerial/drone angle, golden hour | Aspirational ‚Äî "your home could look like this" |
| About section | Real team photo, crew on a roof (safely), owner portrait | Humanizes the company |
| Services | Before/after slider of actual projects | Most powerful conversion tool for roofing |
| Testimonials | Real customer headshot (from Google/Yelp profile) | Social proof feels real |
| Process section | Clean iconography or photos of each step | Reduces anxiety about "what happens next" |
| Gallery | Grid of 6-9 completed projects, mix of styles | Shows range and volume |

**Video that converts:**
- **30-60 second hero background loop** (muted): drone footage of crew working, completed homes, community
- **Customer testimonial video** (60-90 sec): Real homeowner in front of their house
- **Time-lapse of a full roof replacement** (15-30 sec): Satisfying, shareable
- ‚ùå Do NOT autoplay with sound. Ever.

**Photo quality bar:**
- Minimum 1600px wide for hero
- Drone shots of completed work are the #1 differentiator
- Before/after pairs are the #1 conversion driver
- All photos should be color-graded consistently (warm, slightly desaturated)

## 4. CSS & Design Techniques That Elevate Quality

### 4.1 Typography

```css
/* Two-font system: Display + Body */
@import url('https://fonts.googleapis.com/css2?family=Plus+Jakarta+Sans:wght@400;500;600;700;800&family=Inter:wght@400;500;600&display=swap');

/* Display font for headlines */
h1, h2, h3 { font-family: 'Plus Jakarta Sans', sans-serif; }

/* Body font for everything else */
body { font-family: 'Inter', system-ui, sans-serif; }

/* Type scale ‚Äî mobile first */
h1 { font-size: clamp(2.25rem, 5vw, 4.5rem); font-weight: 800; line-height: 1.1; letter-spacing: -0.02em; }
h2 { font-size: clamp(1.75rem, 3.5vw, 3rem); font-weight: 700; line-height: 1.2; letter-spacing: -0.01em; }
h3 { font-size: clamp(1.25rem, 2vw, 1.75rem); font-weight: 700; }
body { font-size: 1rem; line-height: 1.7; color: #374151; }

/* Section labels (small caps above headings) */
.section-label {
  font-size: 0.8125rem;
  font-weight: 600;
  text-transform: uppercase;
  letter-spacing: 0.1em;
  color: var(--accent);
  margin-bottom: 0.75rem;
}
```

**Key differences from template sites:**
- `letter-spacing: -0.02em` on large headings (tighter = more premium)
- `clamp()` for fluid typography (no breakpoint jumps)
- Section labels (small uppercase text above headings) add visual hierarchy
- `line-height: 1.1` on h1 (tight) vs `1.7` on body (readable)

### 4.2 Micro-Animations (Subtle, Not Flashy)

```css
/* Intersection Observer-based scroll reveal */
.reveal {
  opacity: 0;
  transform: translateY(30px);
  transition: opacity 0.6s ease, transform 0.6s ease;
}
.reveal.visible {
  opacity: 1;
  transform: translateY(0);
}

/* Staggered children */
.reveal-stagger > * {
  opacity: 0;
  transform: translateY(20px);
  transition: opacity 0.5s ease, transform 0.5s ease;
}
.reveal-stagger.visible > *:nth-child(1) { transition-delay: 0ms; }
.reveal-stagger.visible > *:nth-child(2) { transition-delay: 100ms; }
.reveal-stagger.visible > *:nth-child(3) { transition-delay: 200ms; }
.reveal-stagger.visible > *:nth-child(4) { transition-delay: 300ms; }
.reveal-stagger.visible > * { opacity: 1; transform: translateY(0); }

/* Animated counter */
.counter { font-variant-numeric: tabular-nums; }

/* Button micro-interaction */
.btn-primary {
  transition: transform 0.2s ease, box-shadow 0.2s ease;
}
.btn-primary:hover {
  transform: translateY(-2px);
  box-shadow: 0 8px 25px rgba(249, 115, 22, 0.35);
}
.btn-primary:active {
  transform: translateY(0);
}
```

```javascript
// Minimal scroll reveal (no library needed)
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('visible');
      observer.unobserve(entry.target);
    }
  });
}, { threshold: 0.15 });

document.querySelectorAll('.reveal, .reveal-stagger').forEach(el => observer.observe(el));

// Animated counter
function animateCounter(el) {
  const target = parseInt(el.dataset.target);
  const duration = 2000;
  const start = performance.now();
  function update(now) {
    const progress = Math.min((now - start) / duration, 1);
    const eased = 1 - Math.pow(1 - progress, 3); // ease-out cubic
    el.textContent = Math.floor(target * eased).toLocaleString();
    if (progress < 1) requestAnimationFrame(update);
    else el.textContent = target.toLocaleString();
  }
  requestAnimationFrame(update);
}
```
